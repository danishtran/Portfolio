## Built with

- [React](https://react.dev/)

- [Material UI](https://mui.com/)
- Deployed with [`gh-pages`](https://github.com/tschaub/gh-pages)

## Running the Project

```
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages (pushes to gh-pages branch)
npm run deploy
```
